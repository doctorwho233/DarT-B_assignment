# 1️⃣ 개념 정리 

## 11.데이터 전처리와 파생변수 생성

```
✅ 학습 목표 :
* 결측값과 이상치를 식별하고 적절한 방법으로 처리할 수 있다.
* 데이터 변환과 가공 기법을 학습하고 활용할 수 있다.
* 모델 성능 향상을 위한 파생 변수를 생성하고 활용할 수 있다.
```

### 11.1. 결측값 처리

- 표시: `:`, `NaN`, `NA`, 빈 문자열 등
- MCAR : 완전무작위
- MAR : 조건부무작위
- NMAR : 비무작위

  
- 제거 : 결측률이 매우 낮거나 분석 가치 없을 때
- 대치
  - 단순 대치 : 평균/중앙값/최빈값/특정값 으로 결측값을 수정
  - 시계열 보간 : `pad`(전값), `bfill`(후값), 선형 보간 등으로 대치
  - 회귀 대치 : 회귀 예측값으로 대치
  - 확률적 회귀 대치 : 회귀대치 + 잡음추가
  - 다중 대치(MI/MICE) : 가상데이터 기반 대치
  - 
### 11.2. 이상치 처리


- EDA/시각화: 분포 확인/ 주로 box plot 활용
- 통계 기준:
   Z-점수: 평균으로부터 ±n시그마
  중앙값: 이상치에 강함
  IQR 기준: Q1−1.5IQR 미만, Q3+1.5IQR 초과 (분포에따라 곱하는 수 변화하기도 함)


- 제거(Trimming): 단순하지만 분산·편향 영향 줄 수 있음
- 값 수정: 대치 파이프라인 적용


### 11.3. 변수 구간화

- 연속형 -> 범주형으로 변환

- 동일 폭 : 값 범위를 동일 간격으로
- 동일 빈도 : 관측치 수가 같도록
- 평활화 : 평균값, 중앙값, 경계값으로
- WOE/IV : 로지스틱 맥락의 정보량 지표


### 11.4. 데이터 표준화와 정규화 스케일링

표준화 : 평균 0, 표준편차 1 인 정규분포로 맞추는 것

정규화 : 범위를 0부터 1로 맞추는 것

RobustScaler: 중앙값, IQR 사용해서 이상치 영향 최소화 시키는 것
